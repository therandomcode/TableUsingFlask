Where did this data come from?

This is an older version of data from the Wikipedia page called The List of Highest Astronomical Observatories. The list has since been updated to correct the first entry, which I have been referring to as the Asian Telescope. Apparently it belongs to the University of Tokyo. The altitude has also been corrected.

Who has access to it?

This table is not only viewable by anyone in the world with an internet connection, but anyone in the world with an internet connection can contribute to this "unfinished list" as well.
